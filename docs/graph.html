Getting Started¶

Installation¶

  1. Get the code. Either clone/fork/branch from GitHub using git or unzip your archive.
git clone https://github.com/cthoyt/pybel.git
  1. cd into your directory
cd pybel
  1. Install with pip. If you want to make changes, add -e before the dot
pip install .
  1. Check that all tests are passing
tox

Basic Usage¶

Command Line Usage¶

PyBEL automatically installs the command pybel. This command can be used to easily compile BEL documents and convert to other formats. See pybel --help for usage details. This command makes logs of all conversions and warnings to the directory ~/.pybel/.

Load, compile, and export to Cytoscape format:

$ pybel to_graphml --path ~/Desktop/example.bel --output ~/Desktop/example.graphml

In Cytoscape, open with Import > Network > From File.

Python API¶

The most useful functions for users are exposed at the top level package. These functions allow for easy import from URL, file, iterable, or a database. It also includes various export options.

>>> import pybel, networkx
>>> g = pybel.from_url('http://resource.belframework.org/belframework/1.0/knowledge/small_corpus.bel')
>>> networkx.draw(g)

pybel.from_database(connection)[source]¶

Loads a BEL graph from a database

Parameters:connection (str) – The string form of the URL is dialect[+driver]://user:password@host/dbname[?key=value..], where dialect is a database name such as mysql, oracle, postgresql, etc., and driver the name of a DBAPI, such as psycopg2, pyodbc, cx_oracle, etc. Alternatively, the URL can be an instance of URL.
Returns:a BEL graph loaded from the database
Return type:BELGraph

Example: >>> import pybel >>> g = pybel.from_database(‘sqlite://’)
